# New Website Bumbu Betutu BR (Reconstruction)

Repository ini berisi kode sumber untuk website Bumbu Betutu BR yang dibangun ulang menggunakan Tailwind CSS untuk kemudahan kustomisasi dan performa tinggi.

## Struktur Folder
- `index.html`: Halaman utama.
- `src/css/`: File CSS kustom (jika ada).
- `public/assets/`: Gambar dan aset statis lainnya.

## Deployment ke Cloudflare Pages
1. Hubungkan repository ini ke akun Cloudflare Pages Anda.
2. Pilih framework preset: "None" (Static HTML).
3. Build command: (Kosongkan).
4. Build output directory: `.` (root).
